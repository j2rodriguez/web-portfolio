Proyecto destacado

Predicción de cáncer mediante modelos de machine learning

Proyecto basado en el dataset Wisconsin Breast Cancer, centrado en la construcción, evaluación y comparación de modelos predictivos para apoyar la detección temprana.

El trabajo presta especial atención a la optimización del umbral de decisión, priorizando la mejora del recall para reducir falsos negativos en un contexto sensible como el diagnóstico clínico.

  • Dataset: Wisconsin Breast Cancer Diagnostic
  • Lenguaje: Python
  • Modelos: Logistic Regression, Random Forest, SVM
  • Métricas: Recall, Precision, F1 Score, ROC AUC
  • Objetivo: mejorar la sensibilidad del modelo en detección temprana
